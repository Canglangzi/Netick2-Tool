# Netick2-Tool
To enhance the Netick user experience and make it more user-friendly, we've provided some simple examples to test all of Netick's features.

These examples include:
MetaVoiceChat support
Car
RPC
OnChanged
Input

This code is for reference only and is not recommended for use in production projects.
